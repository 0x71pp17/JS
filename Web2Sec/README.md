# Endpoint Discovery Tool

A simple JavaScript tool to discover endpoints on a website by scraping HTML links and analyzing JavaScript files.

## Prerequisites
- Node.js (v16 or higher)
- npm (comes with Node.js)


## Setup
1. **Check Node.js and npm versions** (Node.js includes npm):
```bash
node -v
npm -v
```
Both commands should display versions. If not, install Node.js from [nodejs.org](https://nodejs.org), which includes npm.

2. **Install Puppeteer**:
```bash
npm install puppeteer
```

3. The script file is already named `discoverEndpoints.js` in the repository.


## Usage
Run the script with Node.js:
```bash
node discoverEndpoints.js
```

> Replace `'https://example.com'` with the target URL in `discoverEndpoints.js` code befoe running the script.
